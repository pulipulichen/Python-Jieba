# Python-Jieba
A jieba application for desktop user

# Txt格式輸出

各欄位說明：
1. 所有斷詞結果
2. 所有斷詞結果對應的詞性
3. 原本字數
4. 斷詞數量
5. 詞性類型的數量
6. 用詞的entropy：數字越大，表示使用字詞越多元
7. 詞性的entropy：數字越大，表示使用詞性越多元
